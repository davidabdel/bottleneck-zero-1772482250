# Scripts Library — 5 Ready-to-Use Follow-Up Scripts

> Personalize bracketed fields before sending.

## Script 1 — First Response (new inbound)
Hey [Name] — thanks for reaching out about [topic].
Quick context so I can point you the right way: are you mainly trying to improve [outcome A] or [outcome B] right now?

## Script 2 — 2-Hour Nudge (no reply)
Circling back in case this got buried.
If it helps, I can send a quick 3-step plan tailored to [their situation] so you can decide if it’s worth pursuing.
Want me to map that out?

## Script 3 — “Too Expensive” Objection
Totally fair.
Usually this comes down to confidence on outcome + timeline.
If we could [specific measurable improvement] within [timeframe], would this feel like a clear yes?
Happy to outline exactly how that would look in your case.

## Script 4 — “Need to Think About It” Objection
Makes sense — no pressure.
Before you decide, would it help if I sent a one-page breakdown of:
1) what gets implemented first,
2) expected checkpoints in week 1,
3) how to tell quickly if it’s working?

## Script 5 — Re-Engage Warm Lead After Call/DM
Great chatting earlier, [Name].
Based on what you shared, first move I’d make is [specific step].
If you want, I can draft the exact first message/workflow so you can test it this week.
