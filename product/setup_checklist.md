# Setup Checklist (45-Minute Deployment)

## Phase 1 — Foundations (10 min)
- [ ] Define lead channels (IG DMs, X DMs, website form, email)
- [ ] Create a single lead tracking sheet/CRM view
- [ ] Add columns: lead source, first-response timestamp, status, next touch date

## Phase 2 — Script Deployment (15 min)
- [ ] Paste Script 1–5 into your response tool/snippets
- [ ] Customize bracket fields for your niche
- [ ] Add ghost revival templates with scheduled reminders

## Phase 3 — SLA and Automation (10 min)
- [ ] Set first-response SLA target to <=10 minutes
- [ ] Create reminder trigger for untouched leads at +2h
- [ ] Create ghost revival trigger at +24h

## Phase 4 — QA + Go Live (10 min)
- [ ] Test with 2 internal dummy leads
- [ ] Verify link placeholders and CTA text
- [ ] Confirm refund policy copy is present on landing + checkout
- [ ] Record baseline metrics before launch

## Weekly QA
- [ ] Audit 10 recent lead threads for script adherence
- [ ] Track objection categories and update scripts monthly
- [ ] Keep claims compliance-safe (no guaranteed results)
